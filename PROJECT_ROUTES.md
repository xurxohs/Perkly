# Навигация проекта Perkly

В этом документе собраны все страницы фронтенда и эндпоинты бэкенда для удобного ориентирования в проекте.

---

## 🌐 Фронтенд (Next.js)

**Базовый URL локально:** `http://localhost:3000`

### Публичные страницы (доступны всем)

| Путь | Описание |
| :--- | :--- |
| `/` | Главная страница (Лендинг, промо-баннеры, популярные офферы) |
| `/catalog` | Каталог всех товаров, купонов и скидок с фильтрацией и поиском |
| `/offer/[id]` | Детальная страница конкретного товара/оффера (описание, отзывы, цена, кнопка купить) |
| `/auth/login` | Страница авторизации (Вход через Email / Telegram) |
| `/auth/register` | Страница регистрации нового пользователя |

### Страницы для авторизованных пользователей (профиль и покупки)

| Путь | Описание |
| :--- | :--- |
| `/profile` | Личный кабинет пользователя (баланс, бонусы, настройки) |
| `/cart` | Корзина покупок |

---

## ⚙️ Бэкенд API (NestJS)

**Базовый URL локально:** `http://localhost:4000`

*(Протестировать функционал можно через Swagger: `http://localhost:4000/api`)*

### 🔒 Авторизация (Auth)

| Метод | Эндпоинт | Описание |
| :--- | :--- | :--- |
| `POST` | `/auth/register` | Регистрация по email и паролю |
| `POST` | `/auth/login` | Вход и получение JWT токена |
| `POST` | `/auth/telegram` | Авторизация через Telegram Login Widget |

### 👤 Пользователи (Users)

| Метод | Эндпоинт | Описание |
| :--- | :--- | :--- |
| `GET` | `/users/me` | Получить данные текущего авторизованного пользователя |
| `PATCH` | `/users/me` | Обновить профиль (имя, аватар и т.д.) |
| `GET` | `/users/me/stats` | Получить статистику пользователя (кол-во покупок, продаж, отзывы) |

### 🛍️ Офферы / Товары (Offers)

| Метод | Эндпоинт | Описание |
| :--- | :--- | :--- |
| `GET` | `/offers` | Получить список всех офферов (с поддержкой `?search=...&category=...`) |
| `GET` | `/offers/:id` | Получить детальную информацию об оффере по ID |
| `POST` | `/offers` | Создать новый оффер (Только для авторизованных продавцов) |
| `PATCH` | `/offers/:id` | Редактировать оффер |
| `DELETE` | `/offers/:id` | Удалить оффер |

### 💳 Транзакции / Покупки (Transactions)

| Метод | Эндпоинт | Описание |
| :--- | :--- | :--- |
| `GET` | `/transactions` | История покупок текущего пользователя |
| `POST` | `/transactions` | Создать новую транзакцию (купить оффер) |
| `GET` | `/transactions/:id` | Детали конкретной транзакции |
| `PATCH` | `/transactions/:id/status` | Изменить статус покупки (Выполнено, Отменено и т.д.) |

### ⭐ Отзывы (Reviews)

| Метод | Эндпоинт | Описание |
| :--- | :--- | :--- |
| `GET` | `/reviews/offer/:offerId` | Получить все отзывы для конкретного оффера |
| `GET` | `/reviews/offer/:offerId/stats` | Получить средний рейтинг (звезды) и количество отзывов оффера |
| `POST` | `/reviews` | Написать отзыв и поставить оценку (от 1 до 5) |
| `GET` | `/reviews/author/:authorId` | Получить все отзывы, написанные конкретным пользователем |
